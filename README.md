# AI HealthCare - Next.js Project

A digital healthcare consulting platform powered by Artificial Intelligence.

## Features

- **AI Doctor Chat Interface**: Interactive chat where users can describe symptoms and receive AI-powered guidance
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Navigation**: Easy navigation between Home and About Us pages

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and components
  - `page.tsx` - Home page with hero section and chat interface
  - `about/page.tsx` - About Us page
  - `components/` - Reusable React components
    - `Header.tsx` - Navigation header
    - `ChatInterface.tsx` - AI doctor chat component

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- CSS Modules
